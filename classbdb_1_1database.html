bdb::database::database ( const char *  home,
bool  create = false 
)

Opens the database.

If "create" is "true" and database doesn't exist, then creates it.

Exceptions:
bdb::exceptionBDB_ERROR_NOT_FOUND - the database home is not found, or the database doesn't exist.
bdb::exceptionBDB_ERROR_EXISTS - the database already exists (cannot be created).
bdb::exceptionBDB_ERROR_UNKNOWN - unknown error.
Parameters:
[in]homeHome path of the database.
[in]createWhether to create the database if it doesn't exist.
