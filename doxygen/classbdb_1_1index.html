void bdb::index::add_foreign ( table *  foreign,
bool  cascade = true 
)

Adds a foreign key to specified table.

The foreign key means that any key, inserted into this index, must exist in the foreign table.

Exceptions:
bdb::exceptionBDB_ERROR_UNKNOWN - unknown error.
Parameters:
[in]foreignMaster table.
[in]cascadeWhether a cascade removal should be enabled:
true - key from this index will be deleted automatically when related key in master table is being deleted,
false - it's unable to delete related key in master table until same key is present in the index.
